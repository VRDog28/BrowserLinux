# Welcome to the BrowserLinux wiki!

### Here you can find all the features from BrowserLinux!

**Wiki V1.0.0**

## Quick Start

1. Open the project in your browser (https://vrdog28.github.io/BrowserLinux/)
2. Press:
   - ENTER → Load local FS1 file (You select your downloaded main.fs1 from https://github.com/VRDog28/BrowserLinux/releases)
3. Boot into kernel

## Tutorial

1. Setup your username, password and hostname.
2. Boot into the user terminal!

## Basic Commands

- echo  
type `echo hello world` into your terminal and see what happens!
- date  
type `date` to see your current date
- cd  
type `cd /home/Videos` to head into your Videos!
- touch  
type `touch myfile.txt` to create a file
- ls  
type `ls` to see your newly created file!
- rm  
type `rm myfile.txt` to delete your file!  
tip: If you type ls again you wont see myfile.txt
- mkdir  
type `mkdir myfolder` to create a folder  
tip: You can enter `cd myfolder` to head into your newly created folder!
- pwd  
type `pwd` to see where you are
