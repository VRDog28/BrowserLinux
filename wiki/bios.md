# LAKE BIOS UTILITY

Press B when Lake Firmware is booting to enter the Lake Bios Utility 

![](https://raw.githubusercontent.com/VRDog28/BrowserLinux/main/screenshots/screenshot11.png)

Use the arrow keys and enter to navigate!  

## ❌ Disabling the kernel checksum
This basically disables a sha256 checksum that verifies your kernel  
I wont recommend turning this safety feature off
## 🔁 Reloading the bios
This fully reloads the bios
## ❌ Exiting the bios
Exits the bios menu
## ✅ Recovery mode
You can restore a broken fs1
