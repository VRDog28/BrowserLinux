# LAKE BIOS UTILITY

Press delete on the continue screen to enter the bios utility  

![](https://raw.githubusercontent.com/VRDog28/BrowserLinux/main/screenshots/screenshot7.jpg)

Use the arrow keys and enter to navigate!  

## ❌ Disabling the kernel checksum
This basically disables a sha256 checksum that verifies your kernel  
I wont recommend turning this safety feature off
## 🔁 Reloading the bios
This fully reloads the bios
## ❌ Exiting the bios
Exits the bios menu
## ✅ Recovery mode
You can restore a broken fs1 file from a rcvr file
